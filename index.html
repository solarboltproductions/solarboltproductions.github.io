Hello folks, so I've decided to graphically redesign and overhaul our current alpha build for our website! So in this new beta version, the new UI/UX is completely different and (hopefully) better. The back-end code for the website is much more stable and organized, especially because I built this from the ground up with the help of a neat little framework. In general, responsiveness and animations have become more fluid, efficient, and beautiful. The layout actually changes depending on your device (that took some work).The design of this website is based upon Googles own Material Design concepts. The website really comes alive whether you're scrolling, navigating, or hovering over content. In my opinion, it looks way more awesome than the previous one. (Feedback would be appreciated!) As I have put plenty of my time into this, I expect (hope) that y'all would put some more effort out there. I truly do hope that you like it. Thanks for reading, cheers! Changelog and links below.
